HTML Cheat Sheet:

Tags:

Div: A div tag is used to define a section in your HTML code, this tag acts as a container for other elements so that they're easier to group together when you go to style them in a CSS page.

Headings: There are six different headings that can be used, these can be used to emphasize the importance of the heading. For exapmle <h1> is used as a main heading, being the bigest and most bold out of the six headers that could be used. As the numbers increase the text size gets smaller and so does the emphasis on the header. This means that <h6> is the smallest header that we can make with HTML.

Paragraph<p>: The paragraph tag is used to to define a paragraph/body of text, it always starts a new line on your web page.

Image links: For an image link you need to use a link to the actual image which looks something like <a href= "insert link here"> and then you need to use the image source which should look something like this <img src="insert imgae source" >

Bold Text <strong></strong>: This tag is used to bold text, you cna use it in any body of text.

Italic text<em> </em>: These tags are used to italicize the text that you use inside of them.

Underline Text: This for a tag that can be used in HTML and looks something like this <span style="text-decoration:underline;"> </span>

Comments: Comments in HTML code are more for the coder than the poeple reading the information that you have provided on the web page. these are important pieces of information whether it is about what the code does for the page or mabe it is something that you want to hightlight about the code so that you can go back and add more later. This is what a comment should look like in your code < -- insert comment here -->. To be clear, anything that you write within the comment wont show up on your web page it will only appear in yor lines of code.

Line breaks <br>: Line breaks are used to create spacce between text and moving one line of text below the other. teh unique thing about this tag is that it doesn't need to be closed.

Video: To add a vidoe to your web page you need to first set the height and width of the video, which looks something like this <video width="" height="" controls> then you need to include the video source <source src="vid.source" type="vid.type"> and don't forget to close the video tag with </video>.

CSS Cheat Sheet:

Selectors:

Element Selector: Selects elements based on their element type

ID Selector: Selects elements based on their id attribute

Universal Selector: Selects all elements


Properties:

Values: Specify how those properties should be applied.

Properties: Define the appearance of HTML elements.

Comments:We use /*insert comment here*/ to write a comment into the code that cannot be seen on the webpage.


Box Model:

Content:The content of the box, where text and images appear.

Padding:The space between the content and the border.

Border:The border surrounding the padding.

Margin:The space outside the border, separating it from other elements.


Layout:

Static:Default positioning.

Relative:Positioned relative to its normal position.

Sticky:Like relativepositioning until it reaches a certain point and then it acts like fixed.

Fixed:Positioned relative to the viewport.

Absolute: Positioned relative to the nearest positioned ancestor.


Display:

Block:Takes up all the width available.

inline:Takes up only the necessary width.

Inline-/block:Combines the features of the block and inline elements.


Float:

Left:Flosts the element to the left hand side.

Right:Floats elements to the right hand side.

None:Sets to default so there is no float.
